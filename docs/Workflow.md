# AudioTranscription & SegmentsReader Workflow

This document outlines the end-to-end workflow for transcribing audio into segmented JSON files and subsequently viewing and interacting with those transcriptions.

## Phase 1: Creating the Data (AudioTranscription)

The `AudioTranscription` target is the processing engine. It ingests an audio file, performs transcription and phonetic analysis, and generates a structured JSON file containing the segmented linguistic data.

**Steps:**
1. Run the `AudioTranscription` target/tool.
2. Provide it with the source audio file you wish to transcribe.
3. The tool will process the audio, extract text segments, apply translations (if configured), and annotate characters with Pinyin and IPA.
4. The output will be a `.json` file containing a list of structured `SegmentRecord` objects.

## Phase 2: Interactive Reading (SegmentsReader)

The `SegmentsReader` application is an interactive macOS and iOS reader designed to consume and display the JSON output generated by the transcription phase.

**Steps:**
1. Launch the `SegmentsReader` app.
2. If this is your first time, click "Open JSON" (or use `Cmd+O`) to browse for and select the `.json` file generated in Phase 1.
    - *Note: The application will automatically remember the last opened file and instantly load it on subsequent launches.*
3. Scroll through the list of generated segments. By default, you will see the source text alongside its translation.
4. Click the **Phonetic Grid Toggle** (eye icon) next to any segment to reveal the character-by-character breakdown.

**Interacting with the Phonetic Grid:**
- Click or tap on any phonetic cell in the grid to spawn an enlarged **Pop-up View**. The underlying cell will highlight to help you track your position.
- You can freely drag the pop-up anywhere on the screen; the application will remember your preferred position.
- Use the floating toolbar attached to the pop-up to navigate sequentially across cells and segments (Left, Right, Up, Down, First, Last).
- Alternatively, you can navigate using standard keyboard arrows (`Left`/`Right`/`Up`/`Down`) or `Cmd + Left/Right` to jump to extremes.
- Toggle the IPA visibility specifically for the pop-up by tapping the "IPA" eye icon in the toolbar.
- Close the pop-up by tapping the "Close" button.
